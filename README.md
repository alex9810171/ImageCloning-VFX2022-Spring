## Environment
 - flask
 - gunicorn
 - numpy
 - opencv-python
 - requests

## Instruction to run the code
1. Install Dependecies
```
pip install -r requirements.txt
```

2. Run the program
```
.\RUN.cmd   (for Windows system)
./RUN.sh    (for Linux, Ubuntu system)
```
